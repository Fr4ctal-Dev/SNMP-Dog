# snmp-tool
 A SNMP network visualiser
 
## Running the code
### Python
The backend is written in Python 3.7 and requires following packages:
- `pysnmp`
- `ipaddress`
- `Flask`
- `request`
- `json`

Which can all be installed by typing `pip3.7 install <package>` in the project terminal.
